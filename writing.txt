application native:Un développement natif Windows, aussi appelé logiciel "client lourd", est un programme informatique installé sur les postes des utilisateurs sous Windows. Contrairement au client léger (application web), le traitement des données se fait au niveau de la machine de l'utilisateur.
application hybrid:Une application hybride est une application pour mobiles qui combine des éléments HTML5 sous forme de web application mobile et des éléments d'une application native permettant d'utiliser les fonctionnalités natives des smartphones et d'être distribuée en tant qu'application sur les plateformes d'applications (App Store, Android Market, etc.).
application web:Une application web (ou web app) est une interface web applicative disponible uniquement sur le web et accessible via un navigateur internet. C'est une application qui peut être hébergée en cloud ou sur des serveurs dédiés. Toutes les données sont stockées sur un serveur web.
